# organaizer

Тестовое приложение расписания. Выполнено без использования storyboard. Приложение на основе tableView. Календаль использовал FSCalendar и подключал через cocaoPods. Изначально календарь отображается в компактном виде, но реализована возможность полного отображения при помощи свайпа или кнопки. В качестве базы данных для сохранения использовал Realm. Имеется три различных экрана с ячейками. Заполняются данными со своего контроллера. Имеется возможность последующего редактирования ячеек, а также их удаления по свайпу.
